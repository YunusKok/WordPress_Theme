# Competitor WordPress Themes Gap Analysis

The following is a detailed gap analysis comparing our custom directory/listing platform (ThessNest) against top competitor WordPress themes. It highlights the advanced features present in competitors that are currently **missing** in ThessNest.

## 1. Houzez (Real Estate Theme)
**Features Present in Houzez but MISSING in ThessNest:**
- **Advanced CRM & Lead Management**: A built-in pipeline where landlords/agents can track leads, match inquiries to specific properties, and manage customer relations from the frontend dashboard.
- **Floor Plans**: Specific UI meta fields and frontend integrations for uploading and displaying graphical floor plans with dimensions.
- **360° Virtual Tours**: Native integrations for embedding interactive 360 virtual tours (e.g., Matterport, iSpy360).
- **Property Print / PDF Generation**: A frontend action button allowing users to generate a customized, branded PDF brochure of a listing for offline use.
- **WalkScore Integration**: Connecting to the WalkScore API to automatically show neighborhood walkability ratings based on property coordinates.
- **Custom Search Builder**: Drag-and-drop builder for the frontend search bar, allowing admins to easily add/remove any custom fields to the main search filters.

## 2. WP Rentals (Booking Accommodation Theme)
**Features Present in WP Rentals but MISSING in ThessNest:**
- **Hourly & Nightly Bookings**: ThessNest focuses on minimum stays (days/months). WP Rentals supports highly granular hourly or continuous nightly bookings.
- **SMS Notifications (Twilio)**: Native integration for sending SMS alerts for new bookings and messages directly to the landlord's/tenant's mobile phone.
- **Advanced Extra Guest Pricing**: Dynamic pricing that automatically calculates additional per-night fees when a booking exceeds the base guest limit.
- **Security Deposit Pre-Auth**: Automatically placing a hold (pre-authorization) on credit cards rather than charging the deposit fully upfront.

## 3. Listeo (Directory & Listings With Booking)
**Features Present in Listeo but MISSING in ThessNest:**
- **Claim Listing Functionality**: Allows a business or landlord to "claim" an already existing, admin-generated listing on the platform by submitting proof of ownership.
- **Multi-Vendor Wallet System**: A built-in virtual wallet where users can top up funds, use them to pay for bookings, and landlords can request withdrawals directly from their wallet balance.
- **Coupons & Discount Engine**: Native frontend coupon code support for specific properties or users (e.g., "SUMMER10").
- **Business Hours & Service Menus**: Niche directory features showing what days/hours a property or service is open and specific menu prices.

## 4. Homey (Booking and Accommodation Theme)
**Features Present in Homey but MISSING in ThessNest:**
- **Experience Bookings**: Beyond accommodations, Homey allows hosts to list "Experiences" or activities (like Airbnb Experiences) with separate booking parameters.
- **Automated Invoicing Engine**: Built-in, downloadable PDF invoices with sequential numbering for every booking transaction (we currently have Lease Agreements, but not structured financial invoices).
- **Custom Period Pricing UI**: A specific frontend calendar UI for hosts to click and drag over dates to set high/low season prices easily directly from the calendar view.

## 5. Listdo (Directory & Listings)
**Features Present in Listdo but MISSING in ThessNest:**
- **Advanced Review Criteria**: Breaking down reviews into multiple sub-ratings (Cleanliness, Location, Value, Communication, Check-in) rather than just a single overall 5-star rating.
- **Event Ticketing System**: Directly selling tickets for dates/events alongside accommodation/directory listings.
